# Jifywigs E-commerce Platform

A modern e-commerce platform with services booking system.

## Features

### Client (Next.js)
- Modern React with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- Responsive design

### Server (Express.js)
- RESTful API design
- MongoDB with Mongoose
- JWT authentication
- Role-based access control
